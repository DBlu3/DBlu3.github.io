分析

提示了让我们去学Linux file IO~

如果你不懂这部分的知识，那就老老实实地去学一遍吧=。=

其实挺简单的
fd = 0：标准输入
fd = 1：标准输出
fd = 2：标准错误输出

再结合fd.c里面的内容看，已经很明显了
就是让fd=0，然后我们输入LETMEWIN，他就会输出flag了~
