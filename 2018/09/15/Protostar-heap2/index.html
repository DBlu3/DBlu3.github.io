关于auth变量的命名

当程序运行到auth = malloc(sizeof(auth));会计算变量的大小

c

但是因为命名不规范，所以程序计算的是struct auth *auth的大小

而不是计算struct auth { char name[32]; int auth; };的大小

因此它只有4个字节，但是malloc将它与8字节对齐
